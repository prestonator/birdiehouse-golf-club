# BirdieHouse Golf Club

This is a reservation-style golf simulator bay website.

**Setup by MoltenClaude:**
- Next.js project initialized with App Router.
- No TypeScript, ESLint, or Tailwind CSS.
- Project is under version control with Git.
- Repository: [https://github.com/prestonator/birdiehouse-golf-club](https://github.com/prestonator/birdiehouse-golf-club)

**To Deploy to Vercel:**
1. Go to Vercel (https://vercel.com/).
2. Log in.
3. Import the GitHub repository: `https://github.com/prestonator/birdiehouse-golf-club`
4. Vercel will automatically detect the Next.js project and handle the build and deployment.
